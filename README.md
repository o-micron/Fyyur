# Fyyur

# Install

```
python3 -m venv env
```

```
source env/bin/activate
```

```
pip3 install -r requirements.txt
```

# Run

`createdb toy`

`flask db init`

`flask db migrate`

`flask db upgrade`

`FLASK_APP=app.py FLASK_DEBUG=true flask run`
